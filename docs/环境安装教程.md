# 流浪动物救助与领养管理系统 - 环境安装教程

> 技术栈：Vue 3 + Element Plus + Vite（前端）| Node.js + Express（后端）| MySQL 8.0（数据库）
> 操作系统：Windows 11
> 适用场景：目标电脑尚未安装 Node.js、Git、MySQL、VS Code 或 Chrome 时使用；如果这些环境已经安装并能通过版本命令验证，可直接阅读 `快速启动指南.md`。

---

## 一、安装 Node.js

### 1.1 下载
- 官网：https://nodejs.org/
- 选择 **LTS 版本**（18.x 或 20.x），下载 Windows 64-bit 安装包（.msi）

### 1.2 安装步骤
1. 双击下载的 `.msi` 文件
2. 点击 **Next** → 勾选 **I accept** → **Next**
3. 安装路径保持默认 `C:\Program Files\nodejs\` 即可
4. **重要**：在 `Custom Setup` 页面，点击左下角下拉箭头 → 选择 **"Will be installed on local hard drive"**（确保添加到 PATH）
5. 连续 **Next** → **Install** → **Finish**

### 1.3 验证安装
打开 **命令提示符**（Win+R 输入 `cmd` 回车），输入：
```bash
node --version    # 应显示 v18.x.x 或 v20.x.x
npm --version     # 应显示 9.x.x 或 10.x.x
```

### 1.4 配置国内镜像（加速 npm 下载）
```bash
npm config set registry https://registry.npmmirror.com
npm config get registry   # 确认显示 https://registry.npmmirror.com
```

## 二、安装 Git

### 2.1 下载
- 官网：https://git-scm.com/download/win
- 自动开始下载 64-bit 安装包

### 2.2 安装步骤
1. 双击 `.exe` 文件
2. 所有步骤保持 **默认选项**，一路点 **Next** 即可
   - 特别注意：默认编辑器选择 **Vim** 或 **Nano** 都行，不影响
   - 调整 PATH 环境变量选择 **"Git from the command line and also from 3rd-party software"**（默认已选中）

### 2.3 验证安装
```bash
git --version     # 应显示 git version 2.x.x
```

### 2.4 基础配置（可选，如用 GitHub/Gitee）
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

## 三、安装 MySQL 8.0

### 3.1 下载
- 官网：https://dev.mysql.com/downloads/installer/
- 选择 **mysql-installer-community-8.0.xx.msi**（较大的那个，约 300MB+）
- 页面底部点击 **"No thanks, just start my download"**（无需登录）

### 3.2 安装步骤
1. 双击安装包，选择 **Custom** 安装类型
2. 在 Select Products 页面，展开 **MySQL Servers** → **MySQL Server 8.0.x**，添加到右侧
3. 展开 **Applications** → **MySQL Workbench**（可选，自带的数据库管理工具），添加到右侧
4. 点击 **Execute** 开始安装
5. **配置 MySQL Server**：
   - Config Type：选 **Development Computer**
   - Port：**3306**（默认）
   - Authentication Method：选 **Use Strong Password Encryption (RECOMMENDED)**
   - Root Password：设置密码，例如 `root123456`（**务必记住！**）
   - Windows Service Name：`MySQL80`（默认）
   - 勾选 **Start the MySQL Server at System Startup**
6. 点击 **Execute** → **Finish**

### 3.3 验证安装
打开命令提示符：
```bash
mysql -u root -p
# 输入密码后，进入 mysql> 提示符即成功
# 输入 exit 退出
```

如果提示 `mysql 不是内部命令`，需要手动添加环境变量：
1. 右键 **此电脑** → **属性** → **高级系统设置** → **环境变量**
2. 在 **系统变量** 中找到 **Path** → **编辑** → **新建**
3. 添加：`C:\Program Files\MySQL\MySQL Server 8.0\bin`
4. 确定保存，重新打开 cmd 验证

### 3.4 创建项目数据库
```sql
-- 登录 MySQL 后执行
CREATE DATABASE stray_animal_rescue DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
SHOW DATABASES;   -- 确认数据库已创建
```

## 四、安装 VS Code

### 4.1 下载
- 官网：https://code.visualstudio.com/
- 下载 Windows 64-bit 安装包

### 4.2 安装步骤
1. 双击安装包
2. 勾选 **"添加到 PATH"**（Add to PATH）
3. 其余默认，一路 **Next** → **Install**

### 4.3 安装中文语言包
1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展商店
3. 搜索 **Chinese (Simplified)**，点击 Install
4. 按 `Ctrl+Shift+P` → 输入 `display language` → 选择 **中文**

### 4.4 安装推荐扩展
在扩展商店中搜索并安装：
| 扩展名 | 说明 |
|--------|------|
| **Volar** | Vue 3 语法支持（必装） |
| **ESLint** | 代码规范检查 |
| **Prettier** | 代码格式化 |
| **MySQL** | by Weijan Chen，在 VS Code 内管理数据库 |
| **GitLens** | Git 增强工具 |
| **Auto Rename Tag** | 自动重命名 HTML 标签 |
| **Path Intellisense** | 路径自动补全 |

## 五、安装 Navicat（可选）

### 5.1 替代方案
- 如果不想付费，可以用免费的 **DBeaver**：https://dbeaver.io/download/
- 或者直接用 MySQL 自带的 **MySQL Workbench**
- 或者用 VS Code 的 MySQL 扩展

### 5.2 Navicat 安装
1. 下载安装包（官网有 14 天试用）
2. 安装后新建连接 → MySQL
3. 主机：`localhost`，端口：`3306`
4. 用户名：`root`，密码：安装时设置的密码
5. 测试连接成功即可

## 七、环境检查清单

安装完成后，在目标电脑的 cmd 中依次运行以下命令，确认全部正常：

```bash
node --version        # ✅ 应显示 v18.x+ 
npm --version         # ✅ 应显示 9.x+
git --version         # ✅ 应显示 2.x+
mysql --version       # ✅ 应显示 8.0.x
npm config get registry  # ✅ 应显示 https://registry.npmmirror.com
```

全部通过后，就可以开始搭建项目了！
