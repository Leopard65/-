# 流浪动物救助与领养管理系统 — 项目架构说明

## 技术栈
- **前端**：Vue 3 + Element Plus + Vite + Vue Router + Pinia + Axios
- **后端**：Node.js + Express + MySQL2 + JWT + bcryptjs
- **数据库**：MySQL 8.0

## 模块划分原则
1. **前后端分离**：前端和后端独立部署，通过 API 通信
2. **按功能分模块**：每个功能模块（动物、领养、用户等）前后端都有对应目录
3. **RESTful API**：后端接口遵循 RESTful 规范
4. **统一错误处理**：中间件统一捕获和返回错误
5. **JWT 认证**：无状态认证，支持角色权限控制
