## 部署与交付建议

1. 源码交付：执行 `powershell -ExecutionPolicy Bypass -File .\scripts\package-delivery.ps1`，目标电脑重新 `npm install`、配置 `.env`、初始化数据库并启动。
2. 单端口交付：执行 `powershell -ExecutionPolicy Bypass -File .\scripts\package-delivery.ps1 -BuildFrontend`，目标电脑配置后端后用 `npm start`，访问 `http://localhost:3000`。
3. 演示数据：目标电脑初始化数据库后，可在 `server` 目录执行 `npm run seed`，用于答辩演示。
